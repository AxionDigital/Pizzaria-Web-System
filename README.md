
# 🍕 Pizzaria Web System

Sistema completo de **e-commerce para pizzaria**, permitindo que clientes visualizem o cardápio, personalizem pedidos e realizem compras online.

O sistema também possui uma **dashboard administrativa** para gerenciamento de pedidos, produtos e relatórios de vendas.

Além disso, o projeto pode ser integrado com **bots de atendimento automatizado**, como WhatsApp, para automatizar pedidos e comunicação com clientes.

---

# 🚀 Funcionalidades

## 👤 Área do Cliente

### 📋 Cardápio Digital
- Visualização de pizzas e produtos  
- Fotos, descrições e preços  
- Organização por categorias  

### 🍕 Personalização de Pedidos
- Pizza **Meio a Meio**  
- Escolha de **bordas recheadas**  
- Campo de **observações personalizadas**  

### ⚡ Identificação Rápida
Sistema simples de identificação utilizando:

- Nome  
- WhatsApp  

### 🛒 Carrinho de Compras
- Adição de itens  
- Alteração de quantidades  
- Remoção de produtos  

### ℹ️ Informações do Estabelecimento
- Tempo estimado de entrega  
- Horário de funcionamento  
- Dias de atendimento  

---

# 🛠️ Dashboard Administrativa

Área completa para gestão da pizzaria.

## 📦 Gestão de Pedidos
Visualização em tempo real com status:

- Pendente  
- Preparando  
- Entregue  
- Cancelado  

## 🍕 Gestão de Cardápio

- Adicionar produtos  
- Editar produtos  
- Remover produtos  

## 📂 Gestão de Categorias
Organização do cardápio por tipo:

- Pizzas Clássicas  
- Pizzas Especiais  
- Pizzas Doces  
- Bebidas  

## 📊 Relatórios

O sistema gera automaticamente:

- Faturamento total  
- Quantidade de pedidos  
- Ticket médio  
- Ranking de produtos mais vendidos

# 🤖 Integração com Bot

O sistema pode ser integrado com **bots de atendimento automatizado**, como:

- Bot de **WhatsApp**  
- Bot de **Telegram**  
- Assistente virtual no site  

### Possibilidades

- Receber pedidos automaticamente  
- Enviar status do pedido  
- Notificar quando o pedido estiver pronto  
- Responder dúvidas dos clientes  
- Enviar link de acompanhamento do pedido  

---

# 🔄 Fluxo de Funcionamento

1. O cliente acessa o site  
2. Navega pelo cardápio  
3. Escolhe os produtos e personaliza o pedido  
4. Informa **Nome e WhatsApp**  
5. O pedido é enviado para a **dashboard administrativa**  
6. O administrador gerencia o status do pedido  
7. O sistema registra os dados e gera **relatórios automaticamente**
